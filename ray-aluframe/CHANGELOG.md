# 版本记录

## 0.1.0 - 2026-07-15

首个正式版本节点，覆盖从参考图理解到清单、校验和交互预览的完整流程。

### 已具备

- 根据用途、尺寸、载荷和参考图整理铝型材结构方案。
- 生成型材下料、五金搭配、板材固定、加工和装配清单。
- 使用内置通用产品库和建恩铝业供应商数据，不依赖外部网站才能完成清单。
- 输出可旋转、可切换视角、可点选构件和五金的三维交互预览。
- 支持参考图叠加校对，并区分看得见、推断和用户指定的信息。
- 修正参考图识别中的左右、前后方向映射，统一结构数据与预览方向。
- 对跨度、稳定性、移动脚轮和高风险用途进行基础检查。

### 版本说明

- 本版本是后续改动的稳定基线。
- 生成出来的案例文件保留在本地输出目录，不作为 Skill 本体的一部分提交。
